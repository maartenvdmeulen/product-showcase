# product-showcase

## Build Setup

```bash
# install dependencies
$ npm install

# set env variables
API_KEY=
API_ENDPOINT=

# save images local
# 1. create folder in 'static' map 'products'
# 2. pull images
$ node pullImages.js

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
